# SEMHub 内容风格指南

## 写作原则

1. **面向初学者友好**：尽量避免未经解释的专业缩写
2. **结构清晰**：使用 h2/h3 组织文章层级
3. **实例驱动**：尽量配合实际 SEM 图像或示意图
4. **中英对照**：首次出现的专业术语附英文原名

## 语言风格

- 简洁明了，段落不要太长
- 中文为主，专业术语保留英文
- 使用主动语态，避免晦涩的学术腔
- 必要时使用列表、表格、对比等结构化表达

## 难度分级

| 级别 | 标识 | 适用内容 |
|------|------|---------|
| beginner | 入门 | 零基础概念、基础操作、术语解释 |
| intermediate | 进阶 | 参数优化、技术对比、方法论 |
| advanced | 高级 | 高级技术、故障深层分析、前沿进展 |

## 标签使用

- 标签用于内容检索和关联推荐
- 建议每篇文章 2-5 个标签
- 标签以中文为主，关键术语可保留英文
- 示例：`加速电压`、`SE`、`EDS`、`样品制备`、`图像分析`
