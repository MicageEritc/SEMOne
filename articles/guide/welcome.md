## 什么是 SEMHub？

SEMHub 是一个面向**扫描电子显微镜（SEM）**领域的知识平台。无论你是刚接触 SEM 的学生，还是经验丰富的科研人员，都可以在这里找到有价值的内容。

## 如何撰写文章？

在 `articles/` 目录下创建一个新的 `.md` 文件，例如 `articles/my-article.md`，文件开头使用 YAML frontmatter 定义文章元数据：

```yaml
---
title: "文章标题"
description: "文章描述"
date: "2026-06-26"
category: "knowledge"
tags: ["SEM", "电子显微镜"]
author: "你的名字"
published: true
---
```

## 支持的文章分类

SEMHub 目前支持四种文章分类：

| 分类 | 说明 |
|------|------|
| `guide` | 开始学习 — 面向初学者的入门教程 |
| `knowledge` | 知识库 — 深入的技术原理和理论知识 |
| `case` | 案例中心 — SEM 在各领域的实际应用案例 |
| `tool` | 工具 — 实用计算工具和使用指南 |

## Markdown 功能

SEMHub 支持完整的 Markdown 语法，包括：

- **粗体** 和 *斜体* 文本
- [超链接](https://example.com)
- 代码块（如上所示）
- 图片引用
- 表格
- 列表（有序和无序）
- 引用块

> SEM（扫描电子显微镜）利用聚焦的高能电子束在样品表面扫描，通过电子束与样品相互作用产生的各种信号来成像，是材料科学、生物学和纳米技术研究中不可或缺的工具。
